📦 MBG Logistic System
StatusVersionLicense

Sistem manajemen logistik (MBG Logistic) yang dirancang untuk melacak pengiriman, memanajemen inventaris gudang, dan mengoptimalkan rute distribusi barang secara efisien.

✨ Fitur Utama
🚚 Pelacakan Pengiriman (Tracking): Memantau status lokasi barang.
🏢 Manajemen Inventaris: Pencatatan stok barang di gudang.
📊 Dashboard Laporan: Menampilkan statistik pengiriman.
🔒 Sistem Keamanan Akses: Login dengan role Admin dan Staff.
📸 Tangkapan Layar (Screenshot)
Catatan: Berikut adalah tampilan halaman Dashboard Admin dari sistem logistik ini.

📂 Struktur Folder
Proyek ini mengadopsi struktur arsitektur yang terorganisir untuk mempermudah pengembangan:

text

/mbg-logistic
├── assets/             # Gambar, ilustrasi, dan stylesheet CSS/JS statis
├── config/             # Konfigurasi koneksi ke Database
├── controllers/        # Logika aplikasi dan routing backend
├── views/              # Halaman antarmuka pengguna (Frontend)
├── database.sql        # File backup struktur tabel database
├── README.md           # File informasi proyek
└── LICENSE             # Hak cipta dan lisensi
🚀 Panduan Instalasi
Clone repository: git clone https://github.com/pinnpalenn/mbg-logistic.git
Konfigurasi file database di folder config/.
Import file database.sql ke dalam MySQL / DBMS pilihan Anda.
Jalankan aplikasi di web server (seperti XAMPP/Node/dll).
